# 旅遊小幫手 - 設定說明

## 第一步：建立 Supabase 帳號和資料庫

1. 前往 https://supabase.com 免費註冊
2. 點 "New Project" 建立新專案（選擇離你最近的地區，例如 ap-northeast-1 日本）
3. 記下你的 **Project URL** 和 **anon/public key**（在 Settings > API 裡面）

## 第二步：建立資料庫結構

1. 進入你的 Supabase 專案
2. 左側選單點 **SQL Editor**
3. 把 `supabase_schema.sql` 的全部內容貼上去
4. 點 **Run** 執行

## 第三步：設定 APP 連線

打開 `lib/supabase.ts`，把這兩行替換成你的資料：

```ts
const SUPABASE_URL = 'https://你的專案ID.supabase.co';
const SUPABASE_ANON_KEY = '你的 anon key';
```

## 第四步：安裝 Expo Go

在你的手機安裝 **Expo Go** APP：
- iOS: App Store 搜尋 "Expo Go"
- Android: Play Store 搜尋 "Expo Go"

## 第五步：啟動 APP

在電腦終端機（CMD 或 PowerShell）執行：

```bash
cd travel-app
npm start
```

畫面會出現 QR Code，用手機的相機（iOS）或 Expo Go（Android）掃描即可。

## 功能說明

| 功能 | 說明 |
|------|------|
| 行程 | 每日時間軸，可新增景點/交通/餐廳等項目 |
| 預訂 | 管理機票、住宿、租車、憑證 |
| 記帳 | 支援多幣別，自動換算 TWD，可按成員篩選 |
| 準備 | 待辦/攜帶/想買三種清單，可勾選完成 |
| 成員 | 管理同行旅伴，最多 6 人 |

## 顏色主題

主色：橄欖綠 (#7C9A6B)，仿照你提供的 APP 設計風格。
